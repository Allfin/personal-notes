# Personal Note

React project for final project submission **membangun aplikasi catatan menggunakan React**

## Feature

- **Display note**
- **Move note item to archive**
- **Add new note**
- **Delete note**

## Installation

Install project with npm

```bash
  cd personal-notes
  npm install
```
